# Wine-Reviews
Analyzing Wine reviews across various countries and states using R.

This project will conduct data exploration on various reviews of wimes from different countries, including their statistical mean and variance, correlation to price, and various methods of sampling to test how strongly the data follows the Central Limit Theorem. 

The dataset, Wine_Reviews.csv, used containts ~150,000 reviews of various wines from different countries and states in the Unites States. The data lists 11 attributes about each wine: the country, a brief description, designation, review points received, price, Province(State), region, variety, and winery for various wines. There are over 150,000 wine instances in this dataset.

Please refer to the R file, wine-reviews-code.R, to view data pre-processing and statistical analyses implemented and the document 'Wine-Reviews Data Exploration' to understand significant statistical conclusions made from this data exploration excercise.
